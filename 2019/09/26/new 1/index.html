初始化：
00000000 00000000 00000000 00000000
赋值：
11001101 1 0 1 0 0111 0 01 0 0011 00010010
Base2 G DB L A Lim P DPL S Type Base
11001101 10100111 00100011 00010010
CD A7 23 12

0x4->0100取第1位就是0
0x5->0101取前2位是01
0x6->0110取第1位是0
0x8->1000取第1位是0
0x9->1001取第1位是1
0xA->1010取第1位是0
0xB->1011取第1位是1
若取的第1位是偶数就是0不变，若是奇数则改为1
